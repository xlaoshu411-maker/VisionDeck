# VisionDeck · 视觉舱舰

<p align="center">
  <b>开箱即用的数据大屏解决方案</b>
</p>

---

**VisionDeck（视觉舱舰）** 是一个开源的数据可视化大屏项目，帮助你快速搭建专业级的数据展示面板。无论是企业运营看板、实时监控中心、还是年度数据汇报，VisionDeck 都能提供优雅、高效的呈现方式。

## ✨ 特性

- 🎨 **精美设计** — 开箱即用的专业视觉风格，支持深色/浅色主题
- 📊 **丰富图表** — 内置常用图表类型，满足多样化的数据展示需求
- ⚡ **高性能** — 流畅的渲染体验，支持大规模数据实时更新
- 🔧 **易扩展** — 组件化架构，方便自定义和二次开发
- 📱 **响应式** — 适配多种屏幕尺寸，从桌面到超大屏无缝切换

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/niuxinyu/VisionDeck.git

# 进入目录
cd VisionDeck

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
